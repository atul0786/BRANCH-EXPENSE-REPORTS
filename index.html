1 · Connect a published Google Sheet

In Google Sheets: File → Share → Publish to web → choose the data tab → CSV → Publish, then paste that link here. Refresh pulls the latest rows from the sheet (your team can edit the sheet directly and it'll show up here).

2 · Upload a file (Excel / CSV)

Expected columns: Date, Branch Name, Status, Amount (Status: Entered / Pending / No Expense). A Category column (Income / Expense / UPI-Godown / UPI-DDM) is optional. Extra columns are kept but ignored in charts.

Click to choose a .xlsx / .xls / .csv file — or drag it here

3 · Add a manual entry

For quick one-off entries without touching the sheet or a file.

4 · Branch / Category / Status master list

Maintain the canonical reference lists here — same idea as the dropdown "Lists" sheet in your Excel file. Any branch saved in the master list will always appear in the Branch Status board and summary table for every month, even if no data has come in for it yet. If a master-list branch has zero Entered and zero No Expense for the selected month (i.e. it never sent any data — 100% Pending), its row is automatically highlighted in pink, same as BHILAI-DURG / WADI(NGP) in your sheet.
